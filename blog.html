Question-1: What are the different ways to select an element in the DOM?

getElementById(): একটি নির্দিষ্ট আইডি কে ধরতে ইউজ করা হয়। উদাহরণ: document.getElementById("IdName")
getElementsByClassName(): নির্দিষ্ট ক্লাস নাম কে ধরতে ইউজ করা হয়। উদাহরণ: document.getElementsByClassName("className")
getElementsByTagName(): নির্দিষ্ট ট্যাগ নাম কে ধরতে ইউজ করা হয়। উদাহরণ: document.getElementsByTagName("div")
querySelector(): query সিলেক্টর ব্যবহার করে শুধু প্রথম class/id/tag কে ধরা যায়। উদাহরণ: document.querySelector(".ClassName")
querySelectorAll(): query সিলেক্টর ব্যবহার করে শুধু সব class/id/tag কে ধরা যায়। উদাহরণ: document.querySelectorAll(".ClassName")

innerHTML: কোন একটি ট্যাগ সিলেক্ট করে তার মধ্যে কি কি এলিমেন্ট আছে সেগুলো দেখা যায়।
innerText: এটি শুধু ট্যাগের ভিতর থাকা text গুলোকে রিটার্ন করবে।

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach a single event listener to a parent element, and then the event is caught when triggered by any child element.
This way, we don't need to attach event listeners to each child element, which is especially useful when elements are dynamically added or removed.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event that is triggered on a child element propagates upwards to its parent elements.
It starts from the target element and moves up through the DOM tree, triggering event listeners on parent elements.
For example, if you click on a button, the event will first be caught by the button element, then its parent element, and so on up the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

1-Creating an element: example::->
let newElement = document.createElement('div'); newElement.textContent = "Hello, World!";

2-Adding an element: You can add a new element using appendChild() or insertBefore().example::->
document.body.appendChild(newElement); // Adds the new element to the body

3-Removing an element: To remove an element, you use removeChild(). example::->
let parent = document.getElementById('parentElement'); let child = document.getElementById('childElement'); parent.removeChild(child); // Removes childElement
